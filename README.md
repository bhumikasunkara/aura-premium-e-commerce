# Aura Premium E-Commerce

A modern premium e-commerce web application built using React, TypeScript, and Vite.

## Features

* User authentication
* Product catalog
* Shopping cart
* Wishlist
* AI Chat Assistant
* Secure checkout
* Admin dashboard
* Responsive design

## Technologies Used

* React
* TypeScript
* Vite
* CSS
* Context API

## Installation

Clone the repository:

```bash
git clone https://github.com/bhumikasunkara/aura-premium-e-commerce.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

## Project Structure

* `src/` – Application source code
* `assets/` – Images and static assets
* `components/` – Reusable React components
* `context/` – Global state management
